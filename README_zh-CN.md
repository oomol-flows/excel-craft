# DataCraft - 智能表格处理工具包

让您更高效地处理电子表格和数据文件的综合工具集。

## DataCraft 是什么?

DataCraft 是一个可视化工具包,帮助您处理电子表格文件(Excel、CSV 和 TSV),无需编写代码或使用复杂的软件。无论您是在清理混乱的数据、分析销售报告,还是为演示准备信息,DataCraft 都提供简单的构建模块,您可以将它们连接在一起完成工作。

把它想象成数据的乐高积木 - 每个工具都专注做好一件事,您可以用不同的方式组合它们,准确完成您需要的任务。

## 这个工具适合谁?

- **商务人士**:经常使用 Excel 文件并希望自动化重复性任务
- **数据分析师**:需要快速清理和准备数据
- **任何人**:希望更高效地处理电子表格而不需要学习编程

## 用 DataCraft 能做什么?

### 读取和导入文件
首先从 Excel 电子表格、CSV 文件或制表符分隔的文本文件中导入数据。工具会自动检测您的文件类型并正确读取。

### 了解您的数据
快速了解电子表格中的内容 - 有多少行和列,每列包含什么类型的信息,识别任何缺失或有问题的数据。

### 清理混乱的数据
修复常见的数据问题,如:
- 删除空行或填充缺失值
- 删除重复条目
- 修复文本格式问题(多余空格、大小写不一致)
- 将数据转换为正确格式(日期、数字、文本)

### 筛选和排序
通过以下方式精确聚焦您需要的内容:
- 仅显示符合特定条件的行
- 选择要处理的特定列
- 按升序或降序排序信息
- 将大型数据集分解为更小、更易管理的块

### 转换和增强
从现有数据创建新信息:
- 添加带计算值的新列
- 将一列拆分为多列(如分离姓和名)
- 合并列
- 重命名列以提高清晰度

### 分析和汇总
从数据中获得洞察:
- 按类别分组信息(如按地区或月份的销售额)
- 计算总计、平均值、计数和其他统计数据
- 创建数据透视表以查看模式
- 对日期范围执行基于时间的分析

### 验证质量
确保数据符合您的要求:
- 检查必填字段
- 验证数据在预期范围内
- 确保格式遵循规则
- 识别和标记问题

### 合并多个文件
合并来自不同电子表格的信息:
- 基于共同列匹配两个文件中的行
- 将相关信息并排合并
- 支持不同类型的连接(保留所有行、仅匹配行等)

### 采样和拆分
处理数据的部分内容:
- 抽取随机样本用于测试
- 将数据拆分为训练集和测试集
- 从大文件中提取代表性子集

### 处理基于时间的数据
处理日期和时间信息:
- 在时区之间转换
- 提取特定部分(年、月、日)
- 计算日期之间的差异
- 创建基于时间的分组

### 应用公式
使用熟悉的类似电子表格的公式来计算新值,无需理解编程。

### 格式化输出
精确控制导出回 Excel 或 CSV 文件时的结果外观。

## 它是如何工作的?

1. **选择起点**:从读取数据文件开始
2. **添加处理步骤**:按您希望的运行顺序连接工具
3. **即时查看结果**:通过可视化预览观察数据在每一步的转换
4. **准备好时导出**:以您需要的格式保存处理后的数据

工作流程中的每一步都以可视化方式显示,使理解发生的事情和进行调整变得容易。

## 实际应用示例

### 示例 1:清理客户列表
从混乱的客户数据库开始 → 删除重复条目 → 填充缺失的电话号码 → 修复文本格式 → 按姓氏排序 → 导出到 Excel

### 示例 2:分析销售数据
导入销售报告 → 筛选仅显示上一季度 → 按销售区域分组 → 计算每个区域的总销售额和客户数量 → 创建摘要报告

### 示例 3:准备演示数据
读取多个数据文件 → 将它们组合在一起 → 仅选择您需要的列 → 按重要性排序 → 删除任何不完整的记录 → 导出为格式化的 Excel 文件
